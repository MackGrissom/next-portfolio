Quantum is a concept website for a robotics and manufacturing company, showcasing how modern web technologies can present industrial businesses with a premium digital presence.

## Project Highlights

- **Modern Industrial Design** — Clean, professional aesthetic suited for tech/manufacturing
- **Next.js Performance** — Server-side rendering for optimal load times
- **Smooth Animations** — Scroll-triggered animations and transitions
- **Responsive Layout** — Fully optimized for all screen sizes

## Technical Implementation

Built on Next.js with React and styled with Tailwind CSS. The project demonstrates how industrial and manufacturing companies can leverage modern web design to elevate their brand presence.
