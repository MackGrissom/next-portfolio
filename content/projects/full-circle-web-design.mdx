Full Circle Web Design is the landing page for a web design agency, showcasing services and portfolio work. The site is currently live and actively serving clients.

## Project Highlights

- **Professional agency presence** — clean, modern design that builds client trust
- **Service showcase** — clear presentation of web design and development offerings
- **Responsive design** — optimized experience across all devices and screen sizes
- **Performance focused** — fast load times and smooth interactions

Built with React, Tailwind CSS, and Node.js. The site features responsive design patterns, optimized images, and a clean component architecture for maintainability.
